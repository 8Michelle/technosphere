В папке data_preprocessing содержится код для первичной обработки данных.

Файл data_overview.ipynb - для ознакомления с данными, а также для перезаписи документов под соответствующими номерами.

Файл extend_query.ipynb - для расширения запросов за счет синонимов.

Код smart_parser.py извлекает текст из документов: отдельно title, meta, headers и остальные текстовые поля. Также он делает нормализацию этого текста. В целях экономии времени данный код запускается в несколько потоков.

index.py строит обратный индекс базы документов. Этот индекс используется для более быстрого ранжирования по запросам.

nn_parser.py парсит и делает необходимую нормализацию (без лемматизации, чтобы не терять семантику) для нейросетевых моделей.


tf-idf_baseline.ipynb - решение на основе TF-IDF.

bm25.ipynb - решение на основе BM25

bm25f.ipynb - BM25 с учетом различных полей документа. Модификация BM25 для пересчета TF.

bm25f_pairs.ipynb - добавлены поисковые расширения, учет биграмм и проверка орфографии.

scoring_with_index.ipynb - ранжирование с помощью индекса документов.


nn.ipynb - решение на основе USE с использованием эмбедингов заголовков. Добавление мета-тегов не дало прироста качества.

nn_2.ipytb - USE с ипользованием эмбедингов заголовков, headers и с расширенными запросами. Для парсинга тегов headers использовался улучшенный парсер пунктуации.
